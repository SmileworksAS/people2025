# Orbdent People2025

A SaaS application for Orbdent, built with FastAPI, React, TailAdmin (free), and PostgreSQL. Deployed on Fly.io at `people2025.orbdent.com`.

## Setup

### Prerequisites
- Node.js 18+
- Python 3.11+
- Fly.io CLI (`flyctl`)
- Git
- Beekeeper Studio
